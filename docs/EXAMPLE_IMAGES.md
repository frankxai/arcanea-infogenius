## üìê Image Sizes and Placement

### Recommended Dimensions

| Image Type | Width | Height | Placement |
|:-----------|:-----:|:------:|:----------|
| Hero Banner | 1200px | 400px | Top of README |
| Logo | 400px | 400px | Feature sections |
| Architecture | 1200px | 800px | Architecture section |
| Banner | 1200px | 400px | Between sections |
| Icons | 100px | 100px | Feature lists |

### Markdown Image Syntax

```markdown
Standard:
![Alt text](docs/images/image.png)

With size:
<img src="docs/images/image.png" width="600">

With link:
[![Alt text](docs/images/thumb.png)](docs/images/full.png)
```

## üé® Design Guidelines for Generated Images

### Color Scheme
- **Primary**: Oracle Red (#C74634)
- **Secondary**: Elemental colors
- **Background**: Dark theme for mystical feel
- **Text**: White with high contrast

### Style Presets

**Executive Presentation:**
```bash
/arcanea-visual "Professional business presentation slide" \
  --guardian=@vision-artist \
  --elemental=wind \
  --style=transcendent \
  --resolution=1920x1080 \
  --audience=executive
```

**Technical Documentation:**
```bash
/arcanea-visual "Technical system architecture diagram" \
  --guardian=@crystal-architect \
  --elemental=earth \

  --resolution=1920x1080 \
  --audience=technical
```
