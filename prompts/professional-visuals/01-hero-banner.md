# Visual 1: Hero Banner

## Purpose
Landing pages, README headers, first impressions

## The Prompt

```
Create a 16:9 hero banner for a visual intelligence platform.

CONCEPT: A luminous command center where data streams transform into beautiful visuals.

ELEMENTS:
- Central holographic display showing an image being generated
- Floating UI panels with technical parameters (aspect ratio, style, quality)
- Soft gradient background: deep purple (#1a1a2e) to midnight blue (#16213e)
- Subtle grid pattern suggesting precision and technology
- Golden accent highlights (#ffd700) on interactive elements

STYLE: Premium tech product, Apple-level polish, minimal but sophisticated
TEXT: "[YOUR PRODUCT NAME]" in clean sans-serif, bottom center
MOOD: Professional capability, quiet confidence, refined power
```

## Why This Works

**First impressions matter.** This visual communicates "premium tool" without being flashy. The command center metaphor positions users as being in control.

Key design decisions:
- **Holographic display**: Modern, tech-forward, aspirational
- **UI panels**: Shows this is a real tool, not abstract art
- **Dark gradient**: Premium feel, reduces visual noise
- **Golden accents**: Signals quality without being gaudy

## Customization

Replace these for your brand:
- `#1a1a2e` → Your dark brand color
- `#ffd700` → Your accent color
- `[YOUR PRODUCT NAME]` → Your actual product name

## Expected Output

A 16:9 image suitable for:
- GitHub README header
- Landing page hero section
- Documentation intro
- Presentation title slide
