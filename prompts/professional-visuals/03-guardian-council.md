# Visual 3: Guardian Council

## Purpose
Feature showcase, agent explanation, capability overview

## The Prompt

```
Create a 1:1 square image showing 5 AI Guardian agents.

LAYOUT: Pentagon arrangement with central connection point

GUARDIANS (clockwise from top):

1. DRAGON FORGE (top)
   - Fire element, orange/red glow (#ff6b35)
   - Bold, angular silhouette
   - Represents: Transformation

2. CRYSTAL ARCHITECT (top right)
   - Earth element, emerald (#4a9759)
   - Geometric, precise silhouette
   - Represents: Structure

3. OCEAN MEMORY (bottom right)
   - Water element, deep blue (#2e86ab)
   - Flowing, intuitive silhouette
   - Represents: Wisdom

4. VOID GAZER (bottom left)
   - Void element, purple (#7b68ee)
   - Cosmic, mysterious silhouette
   - Represents: Innovation

5. VISION ARTIST (top left)
   - Light element, gold (#ffd700)
   - Refined, elegant silhouette
   - Represents: Excellence

CENTER: Small logo connecting all five

STYLE: Abstract/symbolic, not literal characters. Think app icons, not fantasy art.
BACKGROUND: Dark slate (#1e1e2e) with subtle radial gradient
```

## Why This Works

**Users remember visual metaphors.** The Guardians aren't just featuresâ€”they're memorable characters that help users choose the right tool.

Design rationale:
- **Pentagon layout**: Shows equality, no hierarchy, all valid choices
- **Abstract silhouettes**: Professional, not childish fantasy
- **Elemental colors**: Instantly communicate personality
- **Central connection**: Shows they work together

## Customization

- Adjust colors to match your brand palette
- Change "Represents" to your agent specialties
